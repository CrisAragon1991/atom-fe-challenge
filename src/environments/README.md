
# Proyecto Angular - Atom FE Challenge

Este proyecto fue desarrollado utilizando la plantilla proporcionada para el challenge.

## Ejecución local

Para instalar dependencias y ejecutar el proyecto localmente:

```bash
npm install
npm start
```

Esto levantará la aplicación en modo desarrollo. Puedes acceder a ella en `http://localhost:4200` por defecto.

## CI/CD

El proyecto cuenta con integración continua y despliegue automático (CI/CD) usando Render y GitHub Actions. Cada push a la rama `master` ejecuta pruebas y despliega automáticamente a producción en Render.

---
Puedes agregar más variables de entorno en los archivos `environment.ts` y `environment.prod.ts` según tus necesidades.
